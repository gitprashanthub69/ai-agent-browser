# Test Commands Results

This file will be replaced by the real command outputs after you run `test_commands.py` with a valid Anthropic API key set as the `ANTHROPIC_API_KEY` environment variable.

Run locally:

```powershell
$env:ANTHROPIC_API_KEY="sk-ant-..."
python test_commands.py
```

Sample placeholder output is generated when no API key is present (the script uses a local fallback parser).
